# Features

## Status Key:

 - In-Dev (vX): Currently in active development. Where X is the version number it is expected to be released with
 - Backlog: A future feature thats just an idea at the moment

## All Features

| Key                       | Description                                                                                                                              | Status     |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------|------------|
| SetMaxPrice               | Allows the setting of a max price for a campaign so that everyone knows how much to budget and spend for                                 | In-Dev(v1) |
| Draw                      | Allocates everyone a secret santa and lets them know who it is and what their URL is                                                     | In-Dev(v1) |
| NotifyWhenOpen            | When a secret santa status is Ready but not open, users can join and it will automatically notify them when the status changes to ready. | Backlog    |
| NotifyWhenMaxPriceChanges | Notifies all users that the max price of the secret santa has been changed                                                               | Backlog    |
